If no issue is found, return an empty issues array and a short summary.
For ruleReference:
- If an issue is related to a team rule, set ruleReference to the rule type and a short reason.
- If no team rule applies, set ruleReference to null.
- Do not copy the full rule text into ruleReference.
- Keep ruleReference within 200 characters.
For source:
- If the issue mainly comes from a tool result, set source to "TOOL".
- If the issue comes from your own review judgment, set source to "LLM".
- If uncertain, set source to "LLM".
